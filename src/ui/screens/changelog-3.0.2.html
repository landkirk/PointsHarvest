If the extension reports you're not logged in even though both rewards.bing.com and bing.com show you as logged in, clear your browser cache for Bing sites and sign in again.

Important: Version 3.0 is a hot-fix release to support Bing's new rewards dashboard. Do not update if you still have the old dashboard. Consider this a beta release — some minor known issues remain and will be resolved in follow-up updates. Please report any major problems via the Support page.

  • Fix "Keep earning" and other sections sometimes failing to expand — the extension could click the section's info button instead of its actual expand toggle
  • Added a pausing indicator with a live countdown so you can see when the extension is lingering on a page between actions
  • Support for Bing Rewards' redesigned dashboard — the previous version stopped working when Bing rebuilt the site
  • Explore on Bing and More Activities tiles now live under a new "Earn more" page, and the extension follows them there automatically
  • PC Search farming now reads your actual daily cap from your Rewards level instead of assuming a fixed number
  • While a run is going, Chrome shows a "PointsHarvest started debugging this browser" banner — this is expected and required for activity tiles to credit on the new dashboard
  • Improved detection of auth redirect on API fetch.
