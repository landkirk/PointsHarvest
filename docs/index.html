Installation

Get started in minutes

1

Download the latest release

Download the latest and extract it.

2

Load into Chrome

Navigate to chrome://extensions, enable Developer mode (top right), then click Load unpacked and select the extracted folder.

3

Run it

Open rewards.bing.com, make sure you're signed into your Microsoft account, then click the Points Harvest icon in Chrome and hit Run today's searches.

FAQ

Common questions

Is this an official Microsoft or Bing extension?

No.

Points Harvest is an independent tool with no affiliation with Microsoft, Bing, or the Rewards program. It is not endorsed, authorized, or supported by Microsoft in any way.

Will my Bing account get banned?
There is a risk — Microsoft's Rewards Terms of Service prohibit automated point collection. Points Harvest is transparent about this: the extension shows a warning on first launch. It uses human-like randomized timing and does not spoof anything, but no tool can guarantee your account is safe. Use it at your own discretion.
Is it really free? Any catch?
Yes, completely free. No account, no login, no subscription. The extension runs entirely on your device — there is no tracking or analytics of any kind.
How many points can I earn per day?
Points Harvest automates three phases each run: Explore on Bing activity cards, Daily Set tiles, and PC search farming up to your daily cap. Users in the United States can potentially earn around 7,900 points per month through the extension alone. For a full breakdown of every earning category and what the extension automates, see How Many Microsoft Rewards Points Can You Actually Earn?
Does it need my Microsoft password?
No. Points Harvest never asks for or stores your credentials. It operates entirely through your already-signed-in Chrome session on rewards.bing.com and bing.com — the same way you'd use those sites normally.
Does it work in Edge or Firefox?
Other Chromium-based browsers like Brave, Opera, and Vivaldi support unpacked extensions the same way Chrome does. Edge is more complex — it is primarily designed to install extensions from the Chrome Web Store rather than loading them unpacked. Firefox is not supported; it uses a different extension format.
