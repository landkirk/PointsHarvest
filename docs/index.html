Installation

Get started in minutes

1

Download the latest release

Download the latest and extract it.

2

Open Chrome Extensions

Navigate to chrome://extensions and enable Developer mode (toggle in the top right). Chrome may show a warning about developer mode extensions on each launch — this is normal.

3

Load the extension

Click Load unpacked and select the extracted folder. Updating? Remove the existing PointsHarvest entry first, then load the new folder.

4

Run it

Open rewards.bing.com, make sure you're signed into your Microsoft account, then click the PointsHarvest icon in Chrome and hit Run today's searches.

FAQ

Common questions

Is this an official Microsoft or Bing extension?

No.

PointsHarvest is an independent tool with no affiliation with Microsoft, Bing, or the Rewards program. It is not endorsed, authorized, or supported by Microsoft in any way.

Will my Bing account get banned?
There is a risk — Microsoft's Rewards Terms of Service prohibit automated point collection. PointsHarvest is transparent about this: the extension shows a warning on first launch. It uses human-like randomized timing and does not spoof anything, but no tool can guarantee your account is safe. Use it at your own discretion.
Is it really free? Any catch?
Yes, completely free. No account, no login, no subscription. The extension runs entirely on your device — there is no tracking or analytics of any kind. The only network calls it makes are to check for updates.
How many points can I earn per day?
It depends on your account tier, but PointsHarvest completes all "Explore on Bing" activity cards plus PC search queries up to the daily cap. On a typical account that covers the full daily allotment — usually 30–150 points, depending on your region and tier.
Does it need my Microsoft password?
No. PointsHarvest never asks for or stores your credentials. It operates entirely through your already-signed-in Chrome session on rewards.bing.com and bing.com — the same way you'd use those sites normally.
Does it work in Edge or Firefox?
Other Chromium-based browsers like Brave, Opera, and Vivaldi support unpacked extensions the same way Chrome does. Edge is more complex — it is primarily designed to install extensions from the Chrome Web Store rather than loading them unpacked. Firefox is not supported; it uses a different extension format.
