Features

Everything runs automatically

🔍

Explore on Bing Activities

Detects and completes all your "Search on Bing" activity cards, mapping each to the right query automatically.

📋

Daily Set Tiles

Opens daily set activities. Handles simple ones automatically; surfaces quizzes and polls so you can complete them with one click.

📈

PC Search Farming

After activities are done, farms any remaining PC search points until you hit the daily cap.

🧩

Hands-on When It Matters

Quizzes, polls, and puzzles are surfaced for you to complete manually — the extension handles everything else automatically.

⏱️

Human-like Timing

Uses randomized delays to dwell naturally on pages — no robotic patterns.

Installation

Get started in minutes

1

Download the latest release

Download the latest and extract it.

2

Open Chrome Extensions

Navigate to chrome://extensions and enable Developer mode (toggle in the top right).

3

Load the extension

Click Load unpacked and select the extracted folder.

4

Run it

Click the PointsHarvest icon in Chrome and hit Run today's searches. Make sure you're signed into your Microsoft account.
